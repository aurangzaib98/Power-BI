# 📊 Power BI Retail & Sales Reporting Project  
*Data Technician Bootcamp – Week 2 Power BI Portfolio Project*

This project showcases a series of Power BI reports created during Week 2 of a Data Technician bootcamp. The work focuses on transforming raw data into interactive, insightful dashboards using Power Query, DAX, and a wide range of visualisation techniques. The project demonstrates how Power BI can be used to clean, model, and visualise retail and sales data to support data‑driven decision‑making.

---

## 📁 Project Overview

Across multiple guided labs and hands‑on exercises, this project covers:

- Importing and shaping data from Excel and OData feeds  
- Cleaning and transforming datasets using Power Query  
- Creating relationships and building a semantic model  
- Writing DAX calculated columns and measures  
- Designing interactive reports with slicers, filters, and drill‑downs  
- Publishing reports to the Power BI Service  

The final outputs include several published reports demonstrating strong analytical and visualisation skills.

---

## 🧹 Data Preparation & Transformation (Power Query)

Power Query was used extensively to prepare the datasets for reporting. Key transformation skills include:

- Removing errors, blanks, and duplicates  
- Splitting, merging, and formatting columns  
- Changing data types and applying transformations  
- Creating custom columns for improved modelling  
- Combining multiple data sources (Excel + OData feed)  
- Building clean, analysis‑ready tables  

These steps ensured the data model was accurate, consistent, and ready for DAX calculations.

---

## 🧮 DAX Calculations

The project includes both **calculated columns** and **measures** to support deeper analysis, such as:

- Year, Month, and Date breakdowns  
- Profit and margin calculations  
- Conditional logic for categorisation  
- Aggregations for totals, averages, and percentages  

DAX was used to enhance the semantic model and enable dynamic reporting across visuals.

---

## 📈 Visualisations & Interactive Reporting

A variety of visuals were created to explore retail and sales performance:

- **Bar charts** for category and product comparisons  
- **Line charts** for trend analysis over time  
- **Pie charts** for distribution insights  
- **Maps** for geographic sales patterns  
- **Tables and matrices** for detailed breakdowns  
- **Cards and KPIs** for high‑level metrics  

Interactive elements included:

- Slicers for filtering by date, region, product, or category  
- Cross‑highlighting between visuals  
- Drill‑down and drill‑through navigation  
- Report‑level, page‑level, and visual‑level filters  

These features turned static data into an exploratory, user‑friendly reporting experience.

---

## 🌍 Published Reports

All completed reports were published to the Power BI Service as part of the bootcamp workflow. This included:

- Uploading PBIX files  
- Managing workspaces  
- Sharing reports with trainers  
- Understanding refresh and dataset behaviour  

Publishing demonstrated the full end‑to‑end reporting lifecycle.

---

## 🧠 Key Learnings

- Developed a strong understanding of Power Query for data cleaning and transformation  
- Learned how to build a robust data model using relationships and DAX  
- Improved ability to design clear, interactive dashboards  
- Strengthened data storytelling skills by presenting insights visually  
- Gained confidence working with multiple data sources and combining them in Power BI
